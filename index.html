Gameboy Emulator

Developed a fully functional Game Boy emulator capable of running classic titles. Implemented CPU, memory, and PPU (the Game Boy’s rudimentary GPU) emulation from scratch. Features include support for ROM loading and real-time input handling. This project demonstrates strong understanding of low-level systems programming, hardware emulation, and optimization techniques.

3D Boids

This project explores emergent group behavior using the boids algorithm in Unity. Each fish follows a small set of local rules: alignment, cohesion, and separation, without any central control. From these simple interactions, complex and natural-looking schools of fish emerge. The 3D models are free assets, but all movement logic and behavior systems were programmed from scratch to study how coordinated motion can arise from simple autonomous agents.

Ray Marcher

This project implements a real-time ray marcher written in HLSL, designed for efficient rendering of signed distance fields. The system visualizes a Mandelbulb fractal by evaluating an existing SDF function while smoothly animating its structure through changes in fractal power. Although the SDF definition itself was not written by me, all ray marching logic, shading, and animation were implemented from scratch.

Random Math Demos

When I encounter fascinating natural or mathematical patterns, I recreate and study them through code driven visuals. For instance, the cardioid shapes seen on coffee cups can be modeled using light reflection geometry and modular arithmetic, resulting in captivating animations. Similarly, my Julia set project visualizes 2D slices of a 4D fractal, revealing the hidden structure behind its intricate forms.
